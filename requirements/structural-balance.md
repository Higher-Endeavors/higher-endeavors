I want to build a tool into my website based upon the weight lifting principles of structural balance. The tool needs to allow the user to select their master lift and then enter the weight for that life. Once they have entered their weight for the master lift, the tool will calculate the weight for the other lifts based upon the percentages specified in the structural balance principles.

Requirements:

1. The user should be able to select their master lift from a dropdown list of exercises.
2. The user should be able to enter their weight for the master lift.
3. The tool should calculate the weight for the other lifts based upon the percentages specified in the structural balance principles.
4. The tool should display the weight for the other lifts to the user.
5. The user should be able to specify the unit of measurement (kg or lbs) for the weight entered.
6. The user should be able to specify their preferred number of repetitions for the master lift.
7. The tool should remember the user's preferences and use them for future sessions.
8. The user should be able to save the results calculated into a log of past sessions.
9. The tool should provide a graph of the user's progress over time.
10. The tool should provide a log of the user's lifts for future reference.
